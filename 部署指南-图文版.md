## 第一步：上传代码到 GitHub（1分钟）

### 方法A：使用自动脚本（推荐）

1. 双击运行项目目录下的 `deploy-to-cloud.bat`
2. 按提示操作即可

### 方法B：手动上传

1. 访问 https://github.com/new
2. 填写仓库信息：
   - Repository name: `aicost`
   - Description: `智价AI - 智能建筑造价管理系统`
   - 选择 **Public**（公开）
   - **不要勾选** "Add a README file"
   - 点击 **Create repository**

3. 在本地项目目录打开 PowerShell，执行：

```powershell
cd "C:\Users\Administrator\AppData\Roaming\TRAE SOLO CN\ModularData\ai-agent\work-mode-projects\6a0fb65b56c92673ac94d23f\aicost-main"

git init
git add .
git commit -m "Initial commit"

# 替换为你的仓库地址
git remote add origin https://github.com/你的用户名/aicost.git
git branch -M main
git push -u origin main
```

## 第二步：部署后端到 Render（2分钟）

### 2.1 一键部署

1. 访问 https://dashboard.render.com/blueprint
2. 点击 **"Connect a repository"**
3. 选择 GitHub 账号，授权访问
4. 找到 `aicost` 仓库，点击 **Connect**
5. Render 会自动识别 `render.yaml` 配置文件
6. 点击 **Apply** 开始部署

### 2.2 等待部署完成

- 部署过程约 **5-10 分钟**
- 可以在 Dashboard 看到进度
- 部署成功后，记录你的后端地址：
  - 例如：`https://aicost-backend-abc123.onrender.com`

### 2.3 验证后端是否运行

在浏览器访问：
```
https://aicost-backend-xxx.onrender.com/docs
```

如果能看到 API 文档页面，说明后端部署成功！

---

## 第三步：部署前端到 Vercel（1分钟）

### 3.1 导入项目

1. 访问 https://vercel.com/new
2. 点击 **Import Git Repository**
3. 选择 `aicost` 仓库
4. 配置项目：

| 配置项 | 值 |
|--------|-----|
| Framework Preset | **Vite** |
| Root Directory | **frontend** |
| Build Command | `npm run build` |
| Output Directory | `dist` |

### 3.2 配置环境变量

点击 **Environment Variables**，添加：

- **Name**: `VITE_API_BASE`
- **Value**: `https://你的后端地址.onrender.com/api`

> 例如：`https://aicost-backend-abc123.onrender.com/api`

### 3.3 开始部署

点击 **Deploy**，等待约 **2-3 分钟**

部署成功后，Vercel 会显示访问地址：
```
https://aicost-xxx.vercel.app/aicost/
```

## 自定义域名（可选）

### 绑定自己的域名

1. 在 Vercel 项目设置中，点击 **Domains**
2. 输入你的域名，如 `aicost.你的域名.com`
3. 按提示添加 DNS 记录
4. 等待 DNS 生效（通常几分钟到几小时）

## 常见问题

### Q1: Render 服务休眠怎么办？
Render 免费版一段时间不访问会休眠，首次访问可能需要等待 30 秒唤醒。

**解决方案**：
- 绑定信用卡可以设置永不休眠
- 或使用 UptimeRobot 定时 ping 服务保持活跃

### Q2: 如何更新部署？

修改代码后，提交到 GitHub：

```bash
git add .
git commit -m "更新内容"
git push
```

Render 和 Vercel 会自动重新部署！

### Q3: 数据会丢失吗？
Render 的 PostgreSQL 免费版数据会永久保留，不会丢失。

### Q4: 免费额度够用吗？

| 服务 | 免费额度 | 够用吗？ |
|------|----------|----------|
| Vercel | 100GB/月 | ✅ 个人/小团队完全够用 |
| Render Web | 750小时/月 | ✅ 约31天连续运行 |
| Render DB | 1GB | ✅ 可存储数万条数据 |
